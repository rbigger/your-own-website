# Your Own Website (YOW)

A free, distributable kit for professional small business websites.

## Vision

A person with minimal technical skill can download YOW, point it at their own free Sanity and Cloudflare accounts, and have a professional website live at **no ongoing cost** — and a non-technical owner can keep it current entirely through the CMS, never touching source.

## Status

**In Development** — Kit generalization in progress.

See `docs/vision/VISION-YOW-001-distributable-kit.md` for the full vision.

## Stack

- **Static Site Generator:** Astro
- **Content Management:** Sanity CMS
- **Hosting:** Cloudflare Pages
- **Cost:** $0/month on free tiers
